Emergency Situations

In the event of an emergency, it is important to stay calm and call for help immediately. Some common emergency situations include:

  • Heart attack
  • Stroke
  • Anaphylaxis
  • Seizure
  • Choking

For each of these emergency situations, there are specific first aid steps that should be taken. Please see the following links for more information:
